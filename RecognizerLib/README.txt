Approximators:
	-	Эта папка содержит классы, котоые наследуются от интерфейса IApproxPoints.
		Нужны для аппроксимации( приближение ) точек контуров.

		OtherlApproximators:
		-	Эта папка содержит классы, которые реализуют аппроксинацию не типичным образом 
			(не находят контур ,который бы описывал множество точек минимальным или максимальным образом ,а
			 описывают точки как-то по-другому)

		TypicalApproximators:
		-	Эта папка содержит классы, которые реализуют аппроксимацию типичным образом 

CatchСontours:
	-	Эта папка содержит классы, которые наследуются от интерфейса ICatchСontours.
		Нужны для захвата контуров.

Finders:
	-	Эта папка содержит классы, которые наследуются от интерфейса IFindContours.
		Нужны для нахождения контуров(в формате точек).

Helpers:
	-	Эта папка содержит классы, которые делают разные вещи в разных местах программы.
		
PreparersImage:
	-	Эта папка содержит классы, которые наследуются от интерфейсов IAutoPrepareImage и IPrepareImage.
		Нужны для подготовки изображения к дальнейшему распознавания контуров.

		Masks:
			-	Эта папка содержит классы (в папках ,которые находятся эти папки есть классы ), которые
				находят маски для

			Auto:
				-	Автоматическое вычисление маски (Рекомендованных вариант)

			notAuto:
				-	Ручное вычисление маски 

		NotMask:
			-	Эта папка содержит классы, которые используемся для вычисления маасски

		NN (neural network):
			-	Эта папка содержит классы, которые вычисляют маску с помощью нейронной сети.
		
Recognizs:
	-	Эта папка содержит классы, которые наследуются от интерфейса IContursRecogniz.
		Сами же классы используют классы из папок {Approximators,Finders,PreparersImage} 
		и служат больше для объединения результатов работы трех классов из этих папок. 
		Последовательность работы класса из этой папки:
			изображение -> PreparersImage -> Finders -> Approximators -> точки контуров.

Сontracts:
	-	Эта папка содержит интерфейсы и абстрактные классы, которые нужны для развития библиотеки и поддержки старого функционала.
		Содержит:

			forApproximators:
				-	Эта папка содержит интерфейсы, которые используются для классов аппроксинаторов.

			forCatchСontours:
				-	Эта папка содержит интерфейсы, которые используются для классов из папки CatchСontours.

			forFinders:
				-	Эта папка содержит интерфейсы, которые используются для классов искателей контуров.

			forPreparersImage:
				-	Эта папка содержит интерфейсы и абстрактные классы , которые используются для классов подготовителей изображения.
					Содержит в себе, кроме прочего, еще и интерфейс IBasePrepareImage , который необходим для
					поддержки ручного и автоматического создания "маски" для изображения.

			forRecognizs:
				-	Эта папка содержит интерфейсы, которые используются для классов распознавателей контуров.
